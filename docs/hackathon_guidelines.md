# 🌟 概要
Zennで初となる企業・個人対抗ハッカソンで、業務改革につながるAgentic AIを作ろう。

いま、AIは「答える」ツールから、自律的に「動く」エージェントへと進化しています。業務現場の課題を解決するアイデアは、大企業のエンジニアだけが持っているわけではありません。現場を知るすべての人が、変革の担い手になれる時代です。

本ハッカソンは今回、個人部門を新たに設け、企業・個人を問わずより幅広い挑戦者を募ります。 業務の「課題」をAIエージェントでいかに鮮やかに解決するか。その問いに向き合うプロジェクトが集まることで、日本の業務現場にたくさんのヒントと、わくわくするような刺激が生まれることを期待しています。

# 👥 部門
本ハッカソンは2つの部門で開催します。

### 🏢 企業部門
法人に所属する職員によるチーム。業務課題の解決からソリューション開発まで、目的は問いません。

### 👤 個人部門
所属不問（ただし政府機関等に所属する方を除く）。個人またはチームで参加したい方。

# 📏 審査基準
本ハッカソンでは、技術の先進性だけでなく「いかに実務を動かせるか」を重視して評価します。

* **【ビジネスインパクト】**
    * 業務課題を的確に捉えているか、その解決がどれほど大きな価値（効率化・新規性など）を生むかを評価します。
* **【アプローチの有効性】**
    * Agentic AI（エージェント）としての振る舞いや、アーキテクチャが課題解決に対して論理的で有効であるかを評価します。
* **【完成度・実現性】**
    * 成果物が安定して動作するか、また実際の業務への導入を想定した実装コストや運用性が考慮されているかを評価します。
